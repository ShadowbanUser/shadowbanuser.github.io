谷歌Gemini粗体bug修复插件

Created2026-03-15|Updated2026-03-20|技术
|Word Count:166|Reading Time:1mins|Post Views:

新手的第一个 GitHub 仓库，把谷歌gemini网页版字体加粗遇到特殊标点符号时失效的bug覆盖了，还是有必要记录一下的。

最初只是强迫症的一点偏执，结果跟大厂的 Virtual DOM 防篡改机制和 CSP 拦截斗智斗勇了8h，最后靠gemini自己帮我用纯 DOM 节点重写绕过了安全墙。

虽然是vibe coding出来的，但亲手把第五版代码 commit 上去最终成功时的成就感还是溢于言表的。

也希望大家可以给我的GitHub项目一个star，感谢各位支持！
